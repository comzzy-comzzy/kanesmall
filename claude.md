# Project Standards for Simple Shopping Site

## Technology Stack
- Frontend: HTML, CSS, JavaScript (Vanilla)
- Backend: Node.js with Express
- Storage: Browser localStorage for bag state
- API: RESTful endpoints for product management

## Development Guidelines
- Use semantic HTML for accessibility
- Implement responsive design with mobile-first approach
- Follow REST conventions for API endpoints
- Keep code modular and well-commented
- Use modern JavaScript features (ES6+)

## File Structure
- `/public` - Frontend static files
- `/src` - Backend server code
- `/docs` - Project documentation

## Testing
- Manual testing for key user flows
- Browser compatibility testing for modern browsers
